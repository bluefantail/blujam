BluJm

Victoria University

12 - 25 March

Enter Team

The specificest game jam in Wellington.

BluJm is a 48-hour game making competition with some very uniques rules.

  • You must use Java and the ECS100 Library to make your game
  • No external Libraries/Jars/Engines allowed (apart from ecs100.jar)
  • Your game must be presented primarily through the Window provided by the UI class in the ECS100 Library

BluJm starts at 6pm on a Friday with the announcement of the theme, and runs until 6pm Sunday followed by voting and prize giving. Everyone from First-year university students, to seasoned game makers are welcome to sign up and make some games.

Be aware... there are no restrictions on the amount of preparation of your team's code before the event, be we discourage you building for a specific game before the theme is announced at the start of the competition. And while you must use the window provided by the UI class in the ECS100 Library, there are no restrictions on the modifications you can make to the JFrame itself, though we discourage just removing all of it's components.

Beginners vs Veterans.

Beginners

  • Practice creating code without a strict guideline
  • Learn more about game making and team development
  • Test your skills against seasoned programmers

Veterans

  • Practice making good games with basic tools
  • Share your knowledge about game making with like-minded people
  • Test your programming skills in a restricted domain
